How to make bots work?

Unlike regular single players games bots in multiplayer roulette require setting up queues (so that they can awake randomly during the betting round). Learn more about queues and how to enable them. After queues are set up make sure to create some bots (read more) and enable bots in the multiplayer roulette game settings.

Steps to take after enabling / disabling bots
  • Open Maintenance page and scroll down to Queues block.
  • Scroll down to Cache block and click Clear cache button.
  • Scroll down to Queues block and Click Stop queue worker button (if the supervisor is configured properly the queue worker process will restart automatically).
  • Open Multiplayer roulette game page and refresh it.
